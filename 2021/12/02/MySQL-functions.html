Mysql-常用函数

字符函数

concat:连接

substr:截取子串

upper:变大写

lower：变小写

#显示name为首字母大写，其他小写，fr和las用"_"相连
SELECT CONCAT(UPPER(SUBSTR(`first_name`,1,1)),
LOWER(SUBSTR(`first_name`,2)),'_',LOWER(`last_name`)) AS NMAE
FROM employees;

上面的例子用了嵌套

replace：替换

length：获取字节长度

trim:去前后空格

也可以使用去"str" 去掉前后的"aa"

lpad：左填充

rpad：右填充

instr:获取子串第一次出现的索引

数学函数

ceil:向上取整

round：四舍五入,

mod:取模

取模公式： mod(a,b) = a-a/b*b

例如：

mode(-10,-3)=-10-(-10)/(-3)*-3=-10-(-9)=-1
而 ：
mode(-10,3)=-10-(-10)/(3)*-3=-10-(-9)=-1

floor：向下取整

truncate:截断

rand:获取随机数，返回0-1之间的小数

rand():每次执行返回不同 rand(1):加上数字，执行为同一个值

日期函数

now：返回当前日期+时间

year:返回年

month：返回月

day:返回日

date_format:将日期转换成字符

查询奖金不是空的姓和入职时间，时间按x月/xx日 xx年显示

curdate:返回当前日期

str_to_date:将字符转换成日期

curtime：返回当前时间

hour:小时

minute:分钟

second：秒

datediff:返回两个日期相差的天数

monthname:以英文形式返回月

流程控制函数

if(条件表达式，表达式1，表达式2)

如果条件表达式成立，返回表达式1，否则返回表达式2

case情况1

（一般用于由等值的判断，如果是返回值，不要加；）

case 要判断的变量或表达式或字段 when 常量1 then 值1或语句1； when 常量2 then 值2或语句2； … else 值n或语句n； end

case情况2

（一般用于区间判断，条件判断在when后面）

case when 条件1 then 值1 when 条件2 then 值2 … else 值n end

分组函数

分组函数

分类

max 最大值 min 最小值 sum 和 avg 平均值 count 计算个数

  • sum和avg一般用于处理数值型

  • max、min、count可以处理任何数据类型

  • 分组函数都忽略null

  • 都可以搭配distinct使用，实现去重的统计

Count

count(*) 行数，除去全null的行

count(1):统计结果集的行数—增加一列，值都为1

效率：

  • MyISAM存储引擎，count(*)最高
  • InnoDB存储引擎，count(*)和count(1)效率>count(字段)

group by

和分组函数一同查询的字段，要求是group by后出现的字段

如查询每个部门的人数

having

分组后筛选 如：#查询领导编号>102的领导手下最低工资且大于5000的领导编号和最低工资 第一步 需要这个结果再进行筛选，利用having

其他函数

version 当前数据库服务器的版本 database 当前打开的数据库 user当前用户
